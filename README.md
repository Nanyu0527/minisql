# 使用Git管理代码

## 初次使用

命令：

```bash
git init
git config --global user.name "用户名"
git config --global user.email "用户邮箱"
git remote add origin https://gitee.com/zhang-kongcheng/minisql.git
git add .
git commit -m first_version
git push origin master
```

## 提交
```bash
git pull
git push origin master
```
